ZumiGamma : 로스를 이용한 자율주행 응급차량 만들기
==================

Overview
------------------

  ROBOLINK사의 zumi로봇([로보링크주소](https://github.com/leggedrobotics/darknet_ros))을 이용하여 ROS에 기반한 자율주행 로봇을 개발한다. 
  
  **contributors:** 안하일, 홍성우, 김건기, 조정민

  zumi는 다음과같은 부품들로 구성이 되어있다.  

- Raspberrypi zero  
- Zumi board  
- pi cam  
- LCD  
- battery
